01Password Checker

Description

02PDF Watermark

Description

With this project you can watermark any number of pages efficiently with a watermark of your choosing

How does it work? ?

Using the PyPDF2 library, this python script grabs the pdf and watermark of your choosing and merges them. A 'for loop' loops through all of the pages in the file you chose and stamps every page with your watermark. Once it finishes looping through all of the pages and merges them with the watermark it is then saved as an entirely new pdf file. While this script utilizes the command prompt it can be easily adjusted to run as an executable for business purposes.

This is an example of what is generated using a generic watermark and dummy pdf.

06Machine Learning

Description

Jupyter Notebook is a powerful tool to that uses python to create organized data but it can also be used for machine learning. In this project you can input your own data and Jupyter Notebook will predict which Iris flower species you entered.

How does it work? ?

Using Jupyter Notebook, SkLearn, and a data set containing information about Iris flower species we create a test set along with a training set. We are using the training set to show what data we want and use the test set to ensure our algorithm is accurately predict. Once we know the algorithm works to our standards we input our own data and let the algorithm give us an output. In this case our data was made at random to test our prediction model.
